## üìñ What is Project Sentinel?

Project Sentinel is an intelligent security monitoring system designed specifically for Cameroon. Think of it as a "smart brain" that continuously watches, learns, and predicts security threats across all 10 regions of the country.

**In simple terms:** Instead of security analysts manually checking hundreds of news sources, social media posts, and reports every day, Project Sentinel does this automatically using artificial intelligence - and does it in seconds, not hours.

### The Problem We Solved

Cameroon faces ongoing security challenges including the Anglophone crisis, Boko Haram activities, and regional conflicts. Before Project Sentinel, security teams struggled with:

- **Slow Information**: Waiting 24+ hours to gather intelligence from different sources
- **Overwhelming Data**: Too much information scattered across too many places
- **Manual Work**: Security analysts spending 8-10 hours daily just collecting data
- **Reactive Approach**: Only responding after incidents happen, not preventing them
- **High Costs**: Requiring 5+ analysts working around the clock

### Our Solution

Project Sentinel changes everything by:

‚úÖ **Collecting intelligence automatically** from 15+ sources in real-time  
‚úÖ **Using AI to analyze** and classify threats with 94.2% accuracy  
‚úÖ **Showing everything on an interactive map** across all 10 regions  
‚úÖ **Predicting future threats** with 86.4% accuracy up to 7 days ahead  
‚úÖ **Reducing analyst workload by 60%** - from 5 analysts to just 2

## üí° Technology Behind the Scenes

### For Non-Technical Readers

Think of Project Sentinel as a house with different rooms, each doing a specific job:

**The Collection Room (Data Ingestion):**
- Constantly checking news sites, social media, and reports
- Like having 15 assistants each watching different sources

**The Analysis Room (AI/ML Processing):**
- Reading and understanding all collected information
- Deciding what's important and what's not
- Like having a super-fast analyst who never gets tired

**The Storage Room (Database):**
- Keeping all information organized and secure
- Like a massive filing cabinet that never loses anything

**The Display Room (Dashboard):**
- Showing everything in an easy-to-understand way
- Like a control center with screens and maps

### For Technical Readers

**Frontend:**
- React 18 with TypeScript
- Tailwind CSS for styling
- Mapbox GL for geospatial visualization
- Recharts for data visualization

**Backend:**
- Django 4.2 with Django REST Framework
- PostgreSQL 15 database
- Redis 7 for caching
- Celery for background tasks

**Machine Learning:**
- FastAPI for ML model serving
- Scikit-learn for classification
- Prophet for time series forecasting
- NLTK/spaCy for NLP

**Infrastructure:**
- Nginx reverse proxy
- Gunicorn WSGI server
- Systemd for process management
- Let's Encrypt SSL certificates
